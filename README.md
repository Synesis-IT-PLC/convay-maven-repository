# convay-maven-repository
Convay Maven Repository

Using Convay Meet is straightforward, as it's browser based. Head over to [convay.com](https://convay.com) and give it a try. It's scalable and free to use. All you need is a Google, Facebook or GitHub account in order to start a meeting. All browsers are supported!
